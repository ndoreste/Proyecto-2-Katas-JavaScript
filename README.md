# JavaScript Katas Practice

## Descripción

Collection of JavaScript kata exercises focused on logic building, problem solving and core language practice.

## Tecnologías

- JavaScript

## Objetivo

Practicar lógica de programación y fundamentos de JavaScript mediante ejercicios.

## Funcionalidades

- Ejercicios de lógica
- Práctica de sintaxis JS
- Resolución de problemas
- Base para proyectos interactivos

## Aprendizajes

Refuerzo de pensamiento lógico, funciones, estructuras y resolución paso a paso.

## Mejoras futuras

Organizar katas por dificultad, añadir tests y documentar soluciones.
